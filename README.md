Программа для "подготовки" к экзамену.
В файлах fq и fa находятся построчно соответствующие друг другу вопросы и ответы.
Поиск происходит по совпадению введенной в поле поиска информации с имеющейся в файле вопросов. 
Поддерживается поиск по 2000 строкам файла (715 заполнены).

F5 - развернуть
F6 - свернуть
F7 - выход
Enter - поиск и переход к следующему
Esc - закрыть окно результатов
